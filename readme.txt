Configure a Responsive Photo Stack Slider and Insert it in any Page or Post as a Shortcode. Admin Slide
Fields for Title, Text, and Image.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the `photostack-slider` folder to the to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to Admin Panel -> 'PhotoStack Slider' -> fill Slider fields
4. Insert the shortcode [photostack-slider] to display Slider to any page or post
5. Enjoy!
